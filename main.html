Welcome to the Matrix

De bedoeling van dit project is om je meer bij te leren over matrixen en het nut ervan. Er staan 4 tutorials klaar waarin je leert hoe je matrixen vermenigvuldigd, transponeerd, inverteerd en hoe je de determinant van een matrix berekend. Daarnaast zijn er ook oefeningen voor elk onderdeel waar je je kan testen of je de tutorial hebt begrepen.
Op het einde zijn er nog 2 toepassingen waar je het nut van deze bewerkingen in werking ziet. Je kan zien hoe je matrixen kan gebruiken om sleutels te decoderen en om de toekomst van een bepaald systeem te voorspellen.
Je kan zelf kiezen waar je begint. Als je een nieuwe gebruiker bent raden
