Toepassing: Decrypt the wallet

Hill cipher is a polygraphic substitution cipher based on linear algebra. Each letter is represented by a number modulo 26. Often the simple scheme A = 0, B = 1, …, Z = 25 is used, but this is not an essential feature of the cipher. To encrypt a message, each block of n letters (considered as an n-component vector) is multiplied by an invertible n × n matrix, against modulus 26. To decrypt the message, each block is multiplied by the inverse of the matrix used for encryption. The matrix used for encryption is the cipher key, and it should be chosen randomly from the set of invertible n × n matrices (modulo 26).

Decryptie

Decryptie gebeurd op dezelfde manier als encrypyie maar dan met de inverse van de key. om de inverse van de key(de decoder) de berekenen volstaat het hier niet om gewoon de inverse te berekenen van de encoder maar moet er gewerkt worden met de %26 inverse. Die wordt berekend door ipv de adjunct de delen door de determinant de ,adjunct de vermenigvuldigen met de Modular Multiplicative Inverse(MMI) van de determinant en uiteindelijk elk element %26.
opgelet indien je na %26 een negatief getal uitkomt moet je er 26 bij optellen! vb -268 % 26 = -8 =>18

definitie modular multiplicative inverse

Voor twee gegeven gehele getallen ‘a’ en ‘m’, wordt de modulaire multiplicatieve inverse van ‘a’ onder modulo ‘m’ gedefinieerd als een geheel getal ‘x’ zodat. (a*x) %m =1
